PM x Gemini CLI (一)：環境設定與專案準備

發布日期：2025年10月

本篇文章將引導您完成 Gemini CLI 的基礎環境建置，並準備好您的專案工作環境，為後續的深度協作打下堅實基礎。

我們將涵蓋以下要點：

  • 安裝 Node.js，這是 Gemini CLI 的運行基礎。
  • 安裝 Gemini CLI 工具本身。
  • 安裝 Visual Studio Code，並設定必要的擴充功能。
  • 建立專案資料夾，為您的 AI 協作專案做好準備。

步驟一：安裝 Node.js

Gemini CLI 是一個基於 Node.js 的工具，因此在安裝 Gemini CLI 之前，您需要先安裝 Node.js。

請參考 n8n 教學 (一)：本地安裝 Node.js 與 n8n 環境建置 - 步驟一：下載 Node.js 的說明，完成 Node.js 的安裝。

步驟二：安裝 Gemini CLI

Node.js 安裝完成後，您可以使用 npm (Node.js 的套件管理器) 來安裝 Gemini CLI。請根據您的作業系統選擇對應的安裝指令。

Windows

開啟命令提示字元 (CMD) 或 PowerShell，輸入以下指令：

npm install -g @google/gemini-cli

macOS

開啟終端機 (Terminal)，輸入以下指令：

npm install -g @google/gemini-cli

如果遇到權限問題，您可能需要使用 sudo：

sudo npm install -g @google/gemini-cli

安裝完成後，您可以輸入以下指令來驗證 Gemini CLI 是否成功安裝：

gemini -v

如果成功安裝，您會看到類似 1.0.0 的版本號碼回傳。

步驟三：安裝 Visual Studio Code

Visual Studio Code (VS Code) 是一個輕量且功能強大的程式碼編輯器，我們將使用它來編輯和管理我們的專案檔案。

Windows

前往 VS Code 官方網站 下載 Windows 版本的安裝程式，並按照指示完成安裝。

macOS

前往 VS Code 官方網站 下載 macOS 版本的安裝程式，並按照指示完成安裝。

步驟四：建立專案資料夾

首先，在您的電腦上建立一個新的資料夾，用來存放所有與 Gemini CLI 協作相關的資料和文件。建議命名為清晰的名稱如 NHI-payment-guidelines 之類有意義的名稱。

Windows

在您希望建立專案的位置，例如 C:\Users\您的使用者名稱\Documents，右鍵點擊空白處，選擇「新增」->「資料夾」，然後輸入資料夾名稱。

mkdir C:\Users\您的使用者名稱\Documents\gemini-cli-projects

macOS

開啟 Finder，導航到您希望建立專案的位置，例如 ~/Documents，然後右鍵點擊空白處，選擇「新增檔案夾」，然後輸入檔案夾名稱。

mkdir ~/Documents/gemini-cli-projects

步驟五：使用 VS Code 開啟專案資料夾

開啟 Visual Studio Code，然後將您剛才建立的專案資料夾開啟。

Windows

點擊 VS Code 左上角的「檔案」->「開啟資料夾...」，然後選擇您剛才建立的 gemini-cli-projects 資料夾。

或者，您可以在命令提示字元或 PowerShell 中，導航到該資料夾，然後輸入 code .：

cd C:\Users\您的使用者名稱\Documents\gemini-cli-projects
code .

macOS

點擊 VS Code 左上角的「檔案」->「開啟...」，然後選擇您剛才建立的 gemini-cli-projects 檔案夾。

或者，您可以在終端機中，導航到該檔案夾，然後輸入 code .：

cd ~/Documents/gemini-cli-projects
code .

步驟六：安裝 Markdown 渲染外掛

為了更好地預覽和編輯 Markdown 文件 (Gemini CLI 經常會輸出 Markdown 格式的內容)，我們建議安裝一個 Markdown 渲染外掛。這裡以「Markdown Preview Enhanced」為例。

  1. 開啟 VS Code。
  2. 點擊左側活動欄的「擴充功能」圖示 (或按下 Ctrl+Shift+X)。
  3. 在搜尋框中輸入 Markdown Preview Enhanced。
  4. 找到該擴充功能後，點擊「安裝」。

安裝完成後，您可以在編輯 Markdown 文件時，點擊右上角的預覽按鈕，或右鍵點擊文件選擇「開啟預覽」，即可看到渲染後的 Markdown 內容。

本章總結

在本篇文章中，您已經成功：

  • 安裝了 Node.js 和 Gemini CLI。
  • 安裝了 Visual Studio Code，並設定了專案資料夾。
  • 安裝了 Markdown 渲染外掛，提升了文件編輯體驗。
