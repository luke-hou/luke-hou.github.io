Google Antigravity IDE 教學 (五)：綜效實戰 - 打造您的自動化發布團隊

發布日期：2026年1月

Agentic Synergy：PM 的夢幻團隊

歡迎來到教學最終章。我們已經學會了如何教 AI「看 PRD」(Skill) 和「跑流程」(Workflow)。現在，我們要幹大事了。

每個 Sprint 結束時，PM 都要面對一場惡夢：整理幾十張 JIRA tickets，確認哪些做完了，然後寫出一份給客戶看的 Release Notes，還要寫一份給內部看的行銷文案。現在，這一切可以交給您的「自動化發布團隊」。

情境：The Release Captain (發布指揮官)

我們的目標是建立一個自動化流程，它會掃描本週完成的任務，並自動產出對外公告：

Workflow: weekly_release_prep
Step 1
Fetch Tickets: 讀取 Sprint Board 上所有 "Done" 的項目。
Step 2
Call Specialist: 呼叫 ticket_miner 將生硬的技術票轉為「用戶價值」。
Step 3
Draft Copy: 呼叫 copy_writer 撰寫吸睛標題與行銷文案。
Step 4
Publish: 生成 Release_Notes.md 並發送 Slack 通知。

第一步：培養專家 (Skills)

首先，我們需要兩位專家。請建立以下 SKILL.md：

1. 價值挖掘者 (Expert #1)

路徑: .agent/skills/ticket_miner/SKILL.md

---
name: ticket_miner
description: 從 JIRA/Trello 卡片內容中，分析出對使用者的真實價值 (User Value)。
---
# Role
你是一位資深 PO，擅長將功能點 (Features) 轉化為用戶利益 (Benefits)。

# Instructions
1. 讀取 Ticket 描述 (e.g., "Refactor DB schema for user table").
2. 思考這對用戶有什麼好處？(e.g., "登入速度提升 50%").
3. 輸出簡潔的價值主張 (Value Proposition)。
2. 行銷文案手 (Expert #2)

路徑: .agent/skills/copy_writer/SKILL.md

---
name: copy_writer
description: 將生硬的功能描述，改寫為充滿吸引力的行銷文案。
---
# Role
你是一位曾任職於 Apple 的產品行銷專家，擅長簡潔有力、激勵人心的文字風格。

# Instructions
1. 讀取 `ticket_miner` 的分析結果。
2. 為每個功能撰寫一個 Catchy Headline。
3. 使用 Emoji 增加活潑感✨。

第二步：制定流程 (Workflow)

串聯這兩位專家的流程 .agent/workflows/release_prep.md：

---
description: 準備發布公告的標準流程
---
1. 收集工單 (Collect Tickets)
   - 讀取 `sprint_export.csv`。

2. 挖掘價值 (Mine Value)
   - 對每一張 Ticket，執行 `ticket_miner`。
   - 彙整為 `value_summary.json`。

3. 撰寫文案 (Draft Copy - Synergy Moment)
   - 讀取 `value_summary.json`。
   - 執行 `copy_writer` 產生最終公告內容。
   - 儲存為 `Release_v1.0.md`。

綜效展現

當你運行這個 Workflow 時，原本枯燥的技術清單瞬間變成了行銷素材：

這就是 Synergy。Workflow 確保了每個步驟都不漏掉，而 Skill 確保了產出的內容具有人類專家的水準。

可視化監控：Agent Manager

當這條「生產線」在背景執行時，您不需要一直盯著終端機。Antigravity 提供了強大的 Agent Manager 介面：

  • 按下 Ctrl + M (或點擊右下角圖示) 開啟 Manager Surface。
  • 切換到 "Active Tasks" 分頁。
  • 您會看到 weekly_release_prep 正在執行，並即時顯示當前步驟 (e.g., "Step 2: Mining Value...")。
  • 點擊任務可查看詳細 Log，甚至在 Agent 卡住時進行「人工介入」(Human-in-the-loop)。

系列完結：您的 Agentic 之旅才正要開始

恭喜您讀完了 Google Antigravity IDE 的完整教學系列！

從最初的安裝設定，到定義專家 (Skills)，再到指揮大局 (Workflows)，您已經具備了成為 AI 時代技術領導者的核心能力。

現在，打開您的 IDE，開始打造屬於您的第一支 AI 團隊吧！
