## 🚀 核心功能與技術亮點

### 1. 永久保存的雲端時光機 (Auto-Backup & Archive)
再也不用擔心 LINE 的「檔案已過期」。
*   **全格式備份**：系統自動透過 Webhook 監聽每一則訊息。得益於 **Supabase** 強大的儲存能力，不管是文字對話、重要的截圖證據，還是現場錄影，通通自動上傳雲端保存。
*   **資安與權限**：採用 **API-First 架構** 與 **Session 權限驗證**，確保只有授權的管理員能存取這些敏感資料。
*   **安心下載**：即使 LINE 原檔失效，你隨時都能回到後台搜尋並下載原始檔案。

### 2. 以人為本的數據呈現 (Human-Centric Data)
數據不該只是冷冰冰的 ID，而是代表著每一位努力的夥伴。
*   **真實身份同步**：系統會自動同步 User Profile，將無意義的 User ID (`U1234...`) 轉換為真實的 LINE 顯示名稱。
*   **行內編輯 (Inline Edit)**：如果名稱不直觀（例如客戶用暱稱），管理員可以透過 **AJAX 行內編輯** 功能直接修改顯示名稱，甚至標記特定帳號（如官方帳號/機器人）排除在統計之外，讓數據更純淨。

### 3. 極致流暢的搜尋體驗 (Instant Search)
為了幫管理者省下每一秒鐘，我在前端體驗（UX）上下足了功夫。
*   **防抖搜尋 (Debounce Search)**：搜尋框加入了 300ms 的防抖設計。你不需要按 Enter，輸入關鍵字的瞬間，系統就開始在背景為你過濾成千上萬筆對話。
*   **多維度篩選**：支援「日期範圍快選 (Date Range Shortcuts)」、「群組」、「發言者」等多重條件交叉比對。想找出「上週五、客戶 A 在 B 群組說過關於『報價』的內容」？只需幾次點擊。
*   **關鍵字高亮 (Highlighting)**：系統支援自定義關鍵字字典（如 `Bug`, `Urgent`, `Error`）。當這些字眼出現時，會在列表中以**紅色粗體**高亮顯示，讓你掃視過去絕不錯過重點。
*   **干擾過濾 (Hide Ignore)**：一鍵隱藏「好的」、「謝謝」、「收到」等無效雜訊，還你一個清爽的閱讀版面。

### 4. 決策者的戰情儀表板 (Visual Analytics)
數據不只能看，還能用來決策。首頁的 **Dashboard** 是我利用 **Chart.js** 打造的視覺化指揮中心。
*   **即時戰況**：登入即見今日訊息總量、待處理工單數。
*   **趨勢分析**：透過折線圖與圓餅圖，將抽象的「感覺最近很忙」轉化為真實的數據：「本週訊息量比上週暴增 50%」。
*   **資源爭取**：這些圖表不是畫好看的，而是 PM 跟老闆爭取人力資源、調整工作分配最強而有力的底氣。
