Gmail 重要郵件自動彙報

別讓帳單、收據、重要通知淹沒在信海裡！這是一個專屬的資訊過濾網，每天定時將最重要的郵件彙報給自己。

緣起：過期的帳單與錯愕的我

每天打開 Gmail，總是數十封未讀郵件等著我。在大量的廣告、社群通知和電子報中，時常夾雜著幾封真正重要的信件，例如：

  • 雲端服務 (DigitalOcean, GCP) 的每月收據或費用警示。
  • 保費續費通知。
  • 信用卡電子帳單與繳費提醒。
  • 電話費帳單。

手動在信海中篩選出這些關鍵資訊，不僅耗時，更可怕的是，一不小心就可能錯過，導致服務中斷或產生額外費用。因此讓我決定花一點點時間，必須建立一個自動化的「郵件秘書」。

技術核心：打造個人化的郵件過濾網

這個 n8n 工作流的目標，就是扮演一個忠實的秘書，每天在我指定的時間，幫我完成郵件的篩選與彙報工作。

它的運作流程如下：

  1. 定時觸發 (Schedule Trigger): 每天晚上 9 點，工作流會自動啟動，開始一天一次的郵件檢查任務。
  2. 讀取未讀郵件 (Gmail Node): 首先，它會登入我的 Gmail，並且只抓取「未讀」的郵件，避免重複處理已經看過的信件。
  3. 關鍵字過濾 (Filter Node): 這是整個流程的大腦。我設定了多組關鍵字，例如「收據」、「帳單」、「重要」、「Apple Developer」等。只有郵件主旨符合這些關鍵字的信件，才能進入下一個階段。這就像一個高度客製化的過濾網。
  4. 格式化報告 (Code Node): 使用 AI 寫了一個 Code 節點，將所有篩選出來的郵件，整理成一份乾淨、易讀的 HTML 格式通知，還加上了 Emoji 分類，例如 🍎 代表 Apple、📢 代表重要通知，讓其一目了然。
  5. 即時推播 (Telegram Node): 最後，這份精美的摘要報告會透過 Telegram Bot，準時推送到我的手機。我只需要花幾秒鐘掃一眼，就能掌握所有重要資訊，無需再親自到信箱裡大海撈針。

這個專案的價值

  • 消除錯過恐懼 確保所有重要的帳單、收據和服務通知都能被即時捕捉，再也不用擔心遺漏關鍵訊息。
  • 奪回專注力 將每日例行的「刷信箱」任務完全自動化，把寶貴的專注力留給更需要創造力的工作。
  • 建立資訊儀表板 將 Telegram 變成一個個人的「重要資訊儀表板」，所有關鍵訊息一目了然，無需在多個 App 之間切換。
