n8n 教學 (三)：申請 Gemini API Key

發布日期：2025年5月

接下來要來申請免費的 AI API

到 https://cloud.google.com/?hl=zh-TW

登入 Google 帳號

點選右上角 Console控制台

點左上

點新增專案

改成 n8nTest，然後點建立。

等待數秒

點一下 ‘選取專案’

確認左上角變成 n8nTest

開一個新網頁，到 https://aistudio.google.com/ 並登入 Google 帳號

直接點選上方的 Get API Key

繼續點選 Create API key

點一下搜尋框，選擇先前建立的 n8nTest 專案

緊接著點選 Create API key in existing project

等待數秒

點選 Copy 並將 API key 保存在安全的地方

返回 n8n 教學列表
這邊請特別注意！使用如果您的 API key 綁定在免費的 (free tier) 的專案中，也就是沒有綁定帳單並啟用，那麼 Gemini 經手的資料就有可能被作為訓練用，如有機敏資訊，並且一定要使用 Gemini 的話，請使用付費方案。
