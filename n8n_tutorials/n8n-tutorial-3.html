n8n 教學 (三)：資料轉換與處理

發布日期：2023年10月26日

接下來要來申請免費的 AI API

到 https://cloud.google.com/?hl=zh-TW

登入 Google 帳號

點選右上角 Console控制台

點左上

點新增專案

改成 n8nTest，然後點建立。

等待數秒

點一下 ‘選取專案’

確認左上角變成 n8nTest

開一個新網頁，到 https://aistudio.google.com/ 並登入 Google 帳號

直接點選上方的 Get API Key

繼續點選 Create API key

點一下搜尋框，選擇先前建立的 n8nTest 專案

緊接著點選 Create API key in existing project

等待數秒

點選 Copy 並將 API key 保存在安全的地方
