n8n 教學 (三)：資料轉換與處理

發布日期：2023年10月26日

在本篇 n8n 教學中，我們將深入探討如何在 n8n workflow 中進行資料的轉換與處理。有效的資料處理是自動化流程成功的關鍵。

1. 資料轉換節點

n8n 提供了多種節點來幫助您轉換資料，例如 Set 節點、Function 節點等。

Set 節點

Set 節點是最常用的資料轉換節點之一，它允許您新增、修改或刪除資料欄位。


// 範例：使用 Set 節點新增一個欄位
{
  "nodes": [
    {
      "node": "n8n-nodes-base.set",
      "parameters": {
        "values": [
          {
            "name": "newField",
            "value": "={{ $json.existingField + ' - transformed' }}"
          }
        ]
      }
    }
  ]
}

2. 資料過濾與條件判斷

使用 If 節點或 Switch 節點可以根據條件來控制 workflow 的流程。

If 節點

If 節點根據條件的真假來決定資料流向哪個分支。


// 範例：使用 If 節點判斷數值大小
{
  "nodes": [
    {
      "node": "n8n-nodes-base.if",
      "parameters": {
        "conditions": [
          {
            "value1": "={{ $json.value }}",
            "operation": "larger",
            "value2": "100"
          }
        ]
      }
    }
  ]
}

3. 陣列操作

n8n 也支援對陣列資料進行操作，例如使用 Split In Batches 節點或 Merge 節點。

Split In Batches 節點

此節點可以將一個包含多個項目的陣列拆分成多個批次進行處理。

透過本篇教學，您應該對 n8n 中的資料轉換與處理有了更深入的理解。在實際應用中，靈活運用這些節點將大大提升您的自動化能力。
