標籤

Qdrant 執行檔下載

我們從官方的 Github 下載 .exe 檔案，請認明左上角紅框處正確。

也就是圖片中的 .zip 檔案 (版號會隨著版次更迭而變動，但這個 zip 檔案應該都是同樣名稱)

https://github.com/qdrant/qdrant/releases

下載完成後，在 C:\ 建立新資料夾 “Qdrant” (如果您的C磁碟空間吃緊，也可以放到 D 磁碟，自行替換)，將檔案解壓縮後放進去

完成圖如下

在檔案總管的網址列點一下，將游標移到最前方輸入 cmd 然後按 Enter，會出現小黑窗並且目錄正確。

輸入 qdrant 後按 Enter

順利的話會出現這個畫面，失敗的話我也沒辦法。

此時此刻會看到資料夾自動長出許多東西

這時候就可以在小黑窗連續按兩次 Ctrl + C

然後關閉小黑窗

# Qdrant Web UI 下載

接著我們需要一個前端介面來操作整個系統

一樣從官方的 Github 下載 Web UI 那個檔案

https://github.com/qdrant/qdrant-web-ui/releases/tag/v0.1.41

將檔案解壓縮後會看到這個

繼續點進去會看到這些，通通複製起來。

到 C:\Qdrant 的第一層，手動建立一個資料夾叫 static，並且點進去

把剛剛複製的檔案通通丟進去

此時再度用同樣方式打開小黑窗並執行 qdrant

按住 Ctrl 然後點那個網址就可以自動打開 Qdrant 儀錶板了 :

http://localhost:6333/dashboard#/welcome

如果恰巧打不開，請把網址從 localhost 改成 127.0.0.1 再試試看 :

(理論上 localhost 跟 127.0.0.1 有點像又不是太像，在項跟不像之間)

http://127.0.0.1:6333/dashboard#/welcome

至此今天的進度就完成了
