n8n 教學 (一)：本地安裝 Node.js 與 n8n 環境建置

發布日期：2025年5月

歡迎來到 n8n 自動化教學系列！

本系列將帶您從零開始，一步步學習如何利用強大的自動化工具 n8n 打造屬於自己的工作流。第一篇文章將專注於基礎環境建置，我們將引導您在個人電腦上完成 Node.js 的安裝，並透過 npx 快速啟動 n8n 服務。

重要提醒： 如果您的 Gemini API key 綁定在免費方案 (free tier) 且未啟用帳單，則 Gemini 處理的資料可能被用於模型訓練。若涉及機敏資訊且必須使用 Gemini，請務必升級至付費方案，或使用地端 LLM，亦或是公司提供的 API。

關於安裝方式

本教學採用 npx 進行安裝，此方式最為簡易、快速，適合初學者或臨時性使用。對於開發者或需要長期穩定運行的場景，官方更推薦使用 Docker 進行部署。(注意：若在公司內部網路環境，請遵循資安規範，未經允許請勿使用 Docker。)

步驟一：下載 Node.js

首先，請前往 Node.js 官方網站下載最新的 LTS (長期支援) 版本安裝檔。LTS 版本提供了最佳的穩定性與相容性。

前往 Node.js 官網

步驟二：安裝 Node.js

執行下載的安裝檔，基本上只需遵循安裝精靈的指示，保持預設選項並點擊「下一步」即可完成安裝。

步驟三：驗證 Node.js 安裝

為了確認 Node.js 是否成功安裝，請以系統管理員權限開啟您的命令提示字元 (CMD) 或 PowerShell，並輸入以下指令來檢查版本號：

node -v

如果成功安裝，您會看到類似 v20.11.1 的版本號碼回傳。

步驟四：啟動 n8n

環境準備就緒後，同樣在命令提示字元中，輸入以下指令來下載並執行 n8n：

npx n8n

npx 會自動抓取最新版的 n8n 並執行。首次執行時，系統會詢問您是否要安裝，請輸入 y 並按下 Enter。

安裝過程需要一些時間，請耐心等候。完成後，n8n 會自動啟動本地伺服器。您可以按下鍵盤上的 o 鍵，它將自動在您的預設瀏覽器中開啟 n8n 的歡迎頁面。

步驟五：設定 n8n 帳號

首次進入 n8n，您需要設定一組管理員帳號。請務必記下您設定的 Email 與密碼，這是未來登入的憑證。其餘的問卷選項可以依個人情況填寫，不影響後續使用。

登入後，您將會看到 n8n 的主畫布，恭喜您，至此已成功建置好本地開發環境！

本章總結

恭喜您完成了 n8n 的基礎環境建置！在本篇文章中，我們學習了：

  • 如何驗證 Node.js 是否安裝成功。
