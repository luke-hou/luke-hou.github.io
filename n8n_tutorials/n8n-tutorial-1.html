n8n 教學 (一)：本地安裝 Node.js 與 n8n 環境建置

發布日期：2025年5月

重要提醒： 如果您的 Gemini API key 綁定在免費方案 (free tier) 且未啟用帳單，則 Gemini 處理的資料可能被用於模型訓練。若涉及機敏資訊且必須使用 Gemini，請務必升級至付費方案。

這邊介紹的是較單純的 npx 安裝。

開發者建議用 Docker 安裝 (裝在公司OA的禁止用 Docker)

1. 下載 Node.js

首先到 Node.js 官方網站下載安裝檔案：https://nodejs.org/zh-tw/download

2. 安裝 Node.js

安裝時，用預設值，一直下一步即可。

3. 確認 Node.js 版本

用管理員權限打開 CMD，輸入 node -v 後按 Enter 確認版本。

4. 安裝並啟動 n8n

在 CMD 畫面輸入 npx n8n。

輸入 y 後按 Enter，靜待安裝完畢。

n8n 安裝完畢，會自動啟動服務，按下 o 後，會自動在瀏覽器打開 n8n 起始頁面。

這邊要注意的是 Email 跟 Password 要記好，其他選項隨意選，不影響使用。

登入後可以看到起始頁面，接下來進到下一個 part : Supabase 帳號的申請。
